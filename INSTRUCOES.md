# Instruções Rápidas

## Primeira vez usando o projeto:

1. **Instalar dependências:**
```bash
npm install
```

2. **Inicializar banco de dados:**
```bash
npm run init-db
```

3. **Iniciar servidor:**
```bash
npm start
```

4. **Abrir no navegador:**
```
http://localhost:3000
```
